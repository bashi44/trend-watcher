# Trend Watcher

## 概要
コーダーの仕事とは少し離れているかもしれないが、『LIVE QUIZ LIFE』の運営を通じて、SNSの運用はWebサイトのアクセス数・売上に大きく影響することを学んだため、SNSのカスタマイズ・自動化は必要な知識だと思い制作。

## 機能
1. Twitterのトレンドを1時間毎に監視し、予め設定したキーワードがトレンド上位29位以内に入ると、LINE Notifyで通知する。
2. トレンド入りしたキーワードを含むツイートを自動投稿する。
3. トレンド入りしたキーワードを含むツイートを50件取得し、その中からフォロワーが500人以上いるユーザーのツイートにいいねをする。

## 今後の改善点
1. いいねをするツイートの判別にフォロワー数500人以上という条件しかないため、ネガティブな内容のツイートにいいねをする可能性がある。予め除外するキーワードを指定してもいいが、機械学習を勉強し、アプリケーション自体がキーワードを識別できるようにする必要がある。
2. 現在は完成までの時間を優先し、キーワードをJSONで管理しているが、次回はPythonでデーターベースを絡めたアプリケーションを開発したい。
